Formation Python Débutant

Maîtrisez les bases du langage de programmation le plus populaire au monde, de l'écriture de scripts à la manipulation de données.

Certification Visée

Attestation de Compétences "Développeur Python Niveau 1"

Durée & Modalité

50 heures, 100% Présentiel (Cours du Soir)

Prérequis

Logique algorithmique de base, aucune expérience en code requise.

Organisme

LES COURS DU SOIR (SAS, Pessac, R.C.S. Bordeaux 984 039 933)

Programme Détaillé

Bloc 1 : Fondamentaux de Python (15h)

  • Variables, types de données et opérateurs
  • Structures de contrôle (if, else, for, while)
  • Écrire et appeler des fonctions

Bloc 2 : Structures de Données (15h)

  • Manipuler les listes et les tuples
  • Travailler avec les dictionnaires et les ensembles
  • Compréhensions de listes pour un code efficace

Bloc 3 : Manipulation de Fichiers & Modules (10h)

  • Lire et écrire dans des fichiers texte et CSV
  • Importer et utiliser des modules de la bibliothèque standard
  • Introduction à la gestion des erreurs

Bloc 4 : Projet Pratique (10h)

  • Atelier : créer un programme d'analyse de données simple
  • Débogage et tests du code
  • Bonnes pratiques de développement en Python
